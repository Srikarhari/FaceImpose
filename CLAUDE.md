## FaceImpose — project instructions

### Goal
- Build a small, well-structured project for face superimposition (“face impose”) experiments and tooling.

### How to work in this repo
- Prefer small, incremental changes with working runs at each step.
- If you need to choose libraries/stack, default to **Python** unless I say otherwise.
- Keep inputs/outputs out of git by default (use `.gitignore` once initialized).
- Add a short `README.md` once the first runnable version exists.

### When adding code
- Include a minimal runnable entrypoint (e.g. `python -m faceimpose ...`) as soon as possible.
- Add clear error messages for missing models/files.
- Keep any model weights / large assets out of the repo; document how to download them.

### Before you finish a task
- Ensure the project runs end-to-end for a simple example.
- Summarize what changed and how to run it.
